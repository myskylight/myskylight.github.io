External Style Sheet

1)With an external style sheet, you can change the look of an entire website by changing just one file!

2)Each page must include a reference to the external style sheet file inside the element. The element goes inside the <head >section:

Example:

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

Internal Style Sheet

1)An internal style sheet may be used if one single page has a unique style.

2)Internal styles are defined within the <style> element, inside the <head> section of an HTML page:

Example:

<head>
<style>
body {
background-color: linen;
}
h1 {
color: maroon;
margin-left: 40px;
}
</style>
</head>

Inline Styles Sheet

1)An inline style may be used to apply a unique style for a single element.

2)To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

3)The example below shows how to change the color and the left margin of a <h1>element:

Example:

<h1 style="color:blue;margin-left:30px;">
This is a heading</h1>
