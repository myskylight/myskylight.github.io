Definition and Usage of Center Tag

  • The <Center>tag is used to center-align text.
  • The <Center> tag is not supported in HTML5. Use CSS instead.

HTML <br>Tag

  • Do not need closing tag

Differences Between HTML and XHTML

  • In HTML <br> do not need closing tag
  • In XHTML <br> need closing tag,like this: <br />.
